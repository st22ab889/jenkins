### GitHub SSO 用户认证 ###
```
使用登录 GitHub 的用户名和密码登录 jenkins。
```

<br/>

### 实现步骤 ###
```
1.登录到 GitHub , 添加相应配置
2.在jenkins中安装 "GitHub Authentication" 插件
3.配置 jenkins。注意: 同样先将授权策略修改为 Anyone can do anything, 原因参考"24 Ldap 用户认证集成-.md"
```
